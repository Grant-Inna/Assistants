Цифровые помощники
Руководство пользователя

Анкетирование

Раздел в Цифровых помощниках, с помощью которого пользователи могут проходить тесты, направленные на диагностику знаний, навыков и компетенций. Она используется во всех цифровых помощниках: ученика, учителя и родителя.

Система позволяет выявлять образовательные дефициты, определять уровень освоения тем и рекомендовать соответствующие верифицированные образовательные материалы для дальнейшего развития.

Какие бывают тесты и для кого

Помощник учителя, Помощник родителя
  • Тесты по самодиагностике — помогают оценить уровень профессиональных и личностных компетенций и выявить зоны роста.
Помощник ученика
  • Тесты по самодиагностике — помогают оценить уровень профессиональных и личностных компетенций и выявить зоны роста
  • Диагностические тесты (для инофонов) — направлены на выявление трудностей в знании русского языка как иностранного
  • Олимпиадные задания — представлены тестами и заданиями из регионального и/или заключительного этапов Всероссийской олимпиады школьников (ВСОШ)
Как создать новое тестирование
  1. Создание заданий — вы добавляете все задания, которые могут использоваться в тесте (например, вопросы с вариантами, задания с вводом ответа и т.д.). У каждого задания указывается, для какого типа тестирования оно предназначено (самодиагностика, диагностика, олимпиады).
  2. Создание теста — из общего списка заданий вы собираете конкретный тест: выбираете нужные задания, задаете название, описание и параметры отображения.
Кто может создавать и редактировать контент?
В административной панели пользователи с ролями «Администратор» и «Методист» могут вручную добавлять и редактировать задания и тесты, выбирать формат заданий и настраивать отображение в Помощниках для разных категорий пользователей. Доступ к работе с разделом предоставляется сотрудникам, указанным в заявке от профильного Департамента Министерства просвещения Российской Федерации, после прослушивания обучающего вебинара.
Этап 1. Создание заданий
  • Перейдите в левом меню в пункт «Анкетирование» — вкладка «Задания»
  • Откроется таблица/список ранее созданных заданий
  • В правом верхнем углу нажмите кнопка «Действия» — «Добавить»
В появившемся окне выберите тип задания из списка:
— Выбрать ответ
— Ввести ответ
— Свободный ответ
— Ответ в табличном виде
— Расположить по порядку
— Вставить пропущенное
— Установить соответствие
— Аудирование
— Ввести ответ заданной длины.
  • Откроется форма создания нового задания выбранного типа
  • Заголовок* — укажите название задания
  • Описание* — напишите текст задания
  • Доступность задания* — выберите, в каком типе тестов это задание может использоваться (диагностические тесты, олимпиады, самодиагностика)
  • Категория пользователя* — выберите, кому показать это задание (законный представитель, обучающийся, педагог). Это определяет, в каких пользовательских интерфейсах будет показано задание. Можно выбрать один вариант или сразу несколько
  • Классы (недоступно для самодиагностики) — укажите учебные классы, для которых предназначено данное задание
  • Предмет (недоступно для диагностических тестов) — выберите предмет, к которому относится задание
  • Год задания* — укажите год, например, 2023
  • Баллы за задание* — укажите, если нужно оценивать выполнение
  • Сложность (недоступно для самодиагностики) — выберите какому уровню сложности соответствует задание (базовая, средняя, углубленная)
  • Этап (доступно только для олимпиады) — выберите к какому этапу олимпиады относится задание (отборочный, школьный, муниципальный, региональный, федеральный, заключительный)
  • Ответ — укажите конфигурацию ответа в зависимости от типа задания
  • Компетенции (актуально только для самодиагностики) — выберите подходящие компетенцию, область диагностики, укажите описание и баллы по данной компетенции
  • Нажмите кнопку «Создать и вернуться к списку» или «Создать и добавить новый»
  • Задание сохранится в статусе «Черновик» — его пока не видят пользователи
  • В списке найдите созданное вами задание, в столбце «Действия» нажмите кнопку с тремя точками «…» и выберите «Опубликовать»
Этап 2. Создание тестирования
  • Перейдите в левом меню в пункт «Анкетирование» — вкладка «Тесты»
  • Откроется таблица/список ранее созданных тестов
  • В правом верхнем углу нажмите кнопка «Действия» — «Добавить»
  • В появившемся окне выберите тип создаваемого тестирования из списка (олимпиады, диагностические тесты, тесты по самодиагностике)
  • Откроется форма создания нового тестирования выбранного типа
  • Заголовок* — укажите название тестирования
  • Описание — напишите краткое пояснение для пользователя о целях теста
  • Инструкция — напишите краткое пояснение для пользователя с инструкцией прохождения тестирования
  • Класс* (недоступно для самодиагностики) — укажите учебные классы, для которых предназначен данный тест
  • Предмет (недоступно для диагностических тестов) — выберите предмет, к которому тестирование
  • Год* (доступно только для олимпиад) — укажите год олимпиады, например, 2023
  • Вид (доступно только для олимпиад) — выберите вид олимпиады (письменная, устная)
  • Этап (доступно только для олимпиад) — выберите к какому этапу олимпиады относится тестирование (отборочный, школьный, муниципальный, региональный, федеральный, заключительный)..
  • Категория пользователя* — выберите, кому показать этот тест (законный представитель, обучающийся, педагог). Это определяет, в каких пользовательских интерфейсах будет показано тестирование. Можно выбрать один вариант или сразу несколько
  • Уровень (доступно только для диагностических тестов) — выберите какому уровню соответствует тестирование (элементарный, базовый, продвинутый)
  • Время на ответе в минутах* — укажите максимальное время на ответ в минутах
Перейдите во вторую вкладку «Выбрать задания»
  • Нажмите кнопку «Добавить»
  • В зависимости от выбранного типа тестирования, Система предложит выбрать подходящие задания
  • Выберите и отметьте нужные задания и нажмите кнопку «Добавить выбранное»
  • Задания появятся в списке заданий в данном тестировании. Вы всегда можете удалить ненужные позиции, а также добавить еще заданий
  • Нажмите кнопку «Сохранить и вернуться к списку»
  • В списке созданных тестов найдите созданный вами тест, в столбце «Действия» нажмите кнопку с тремя точками «…» и выберите «Опубликовать»
Сразу после публикации, тестирование станет доступно в тех Помощниках, которые вы выбрали на этапе создания.
  • Задания создаются один раз и могут использоваться в нескольких тестах
  • Обязательно корректно указывайте Доступность задания, иначе оно не появится в списке при создании теста
  • После создания теста его можно редактировать: менять заголовок, описание и состав заданий
